Ricardo Collado ¿Que es Java?

Java SE Basico

Java, es un lenguaje de programacion orientado a objeto, lo que quiere decir es que, su paradigma se basa en este estilo de programacion.


Para tener un poco de historia. Java es creado por James Gosling en el 1995, hace ya 27 años

A pesar de que fue creado hace ya varios años, Java se ha mantenido actualizandose. Debio a su gran comunidad, y a que es un lenguaje robusto, muchas compañias grandes todavia utilizan esta tecnologia. Lo que ha llevado en varias ocaciones a que Java ajuste su estilo de programacion a las novedades que traen las mas recientes tecnologias. Creando asi la oportudad de que los programadores de java, sigan siendo indispensables para la industria.

¿Que significa que java sea Multiplataforma?

Java utiliza una maquina virtual que le permite correrse donde sea. Sabemos que este curso es introduccion a la programacion. Asi que primero debemos de entender, que el factor multiplataforma es una ventaja, ya que, no muchos lenguajes dominaban esto en el pasado. Aun en la actualidad, debido a su rigidez, muchos lenguajes se limitan en la plataforma.

Pero... ¿que es una plataforma?

Cuando Hablamos de plataforma, estamos hablando del sistema operativo, que podrian ser: Linux, Windows, MacOs, Android, etc.


Muy seguramente ya has visto Java en otros lugares, de hecho... se ha usado Java para programar los reloj que utlizan los microwave. Pero no solo alli, tambien en calculadoras. Y en tu misma computadora, existen aplicaciones que utilizan Java. Y si... muy probblemente si trabajaste para alguna compañia de telecomunicaciones, viste Java... como verizon y otros...

Seguro

Java es seguro, su estructura y forma de crear codigo crean un ambiete seguro y una plataforma segura para las grandes compañias. Utiliza llaves publicas de encriptacion para prevenir codigo nefario o virus que se infiltren en la plataforma.

Orientado a objetos (paradigma)

El proposito general de java es centrarse totalmente en la programacion orientada a objetos. Por eso utiliza sus pilares tan fuertemente, como lo son: La encapsulacion, La herencia, el polimorfismo y las clases.

Network (redes)

Java tiene otros propositos y ventajas. Esta diseñado para trabajar bien con las redes o network. y su libreria extensas (recordemos que una libreria son una serie de caracteristicas (o features) que permiten ser reutilizadas) en redes, permiten un manejo de protocolos de transmision tales como HTTP o FTP.

Multithreading - Multihilo

Primero, que es el multithreading: Eso es el hecho de que un programa tenga la capacidad de correr dos algoritmos al mismo tiempo. Que pueda hacer dos cosas a la vez, sin necesidad de esperar a que una se termine para que suceda la otra. Esto ya es un conceptop muy avanzado, pero no se puede definir Java, sin mencionar esta gran ventaja. Java facilita las operaciones multiples usando todos los cores (nucleos) de nuestro procesador.

EL JVM

El jvm, es una maquina virtual de java, que lo que hace es, tomar el codigo de java, llevarlo a algo conocido como "ByteCode" compilarlo y traducirlo, segun la plataforma en la que este, a algo entendible para ese sistema operativo.
